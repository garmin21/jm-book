:::tip
所谓粘性布局就是当内容容器没有超出时，footer 是紧贴在某一个方向上顶或底的，当内容超出的时候，footer 紧随在内容容器之后，并不会超出容器。

效果：比如一个容器，在内容不超出的情况下，固定在顶部或者顶部。但内容变大超出的时候，跟随文档流移动
:::

:::normal-demo

```css
.outer {
  height: 300px;
}
.inner {
  min-height: 300px;
  padding-bottom: 50px;
  box-sizing: border-box;
}

footer {
  height: 50px;
  background: red;
  margin-top: -50px;
}
```

```html
<div class="outer">
  <div class="inner">
    <div class="main">
      我是内容盒子 <br />
      我是内容盒子 <br />
      我是内容盒子 <br />
      我是内容盒子 <br />
      我是内容盒子 <br />
      我是内容盒子 <br />
    </div>
  </div>
  <footer></footer>
</div>
```
:::
